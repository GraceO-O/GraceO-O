BIR FORM 0619-E

Tax Filing Management System

A desktop-based application developed using Python Tkinter and MySQL for managing and generating BIR Form 0619-E records. The system provides an intuitive user interface that simplifies data entry, validation, and record management. It also supports full CRUD (Create, Read, Update, and Delete) operations, enabling users to efficiently store, retrieve, modify, and maintain taxpayer information and form records.
